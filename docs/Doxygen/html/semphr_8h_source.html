291 /*
292  * xSemaphoreAltTake() is an alternative version of xSemaphoreTake().
293  *
294  * The source code that implements the alternative (Alt) API is much
295  * simpler because it executes everything from within a critical section.
296  * This is the approach taken by many other RTOSes, but FreeRTOS.org has the
297  * preferred fully featured API too. The fully featured API has more
298  * complex code that takes longer to execute, but makes much less use of
299  * critical sections. Therefore the alternative API sacrifices interrupt
300  * responsiveness to gain execution speed, whereas the fully featured API
301  * sacrifices execution speed to ensure better interrupt responsiveness.
302  */
303 #define xSemaphoreAltTake( xSemaphore, xBlockTime ) xQueueAltGenericReceive( ( xQueueHandle ) ( xSemaphore ), NULL, ( xBlockTime ), pdFALSE )
